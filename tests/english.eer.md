## grammar.inspi
```
<sentence>: <subject> <space> <predicate> <space> <object>;

<subject>: <noun_phrase>;
<object>: <noun_phrase>; 
<noun_phrase>: <article> <space> <noun>;

<article>: "the" | "a" | "an";
<noun>: "dog" | "cat" | "automobile" | "man";

<predicate>: <adverb> <space> <verb> | <verb>;
<adverb>: "desperately" | "hardly" | "ferociously";
<verb>: "eats" | "wants" | "gives";

<space>: ' ' | ' ' <space>;
```

## input.txt
```
the cat desperately wants an automobile```
